⚙️ Middlewares / 🧬 Core

  • BasicAuth (⭐37k) - Basic auth middleware provides an HTTP basic authentication. It calls the next handler for valid credentials and 401 Unauthorized for missing or invalid credentials.
  • Cache (⭐37k) - Intercept and cache responses.
  • CORS (⭐37k) - Enable cross-origin resource sharing (CORS) with various options.
  • CSRF (⭐37k) - Protect from CSRF exploits.
  • Encrypt Cookie (⭐37k) - Encrypt middleware which encrypts cookie values.
  • ETag (⭐37k) - ETag middleware that lets caches be more efficient and save bandwidth, as a web server does not need to resend a full response if the content has not changed.
  • Expvar (⭐37k) - Expvar middleware that serves via its HTTP server runtime exposed variants in the JSON format.
  • Favicon (⭐37k) - Ignore favicon from logs or serve from memory if a file path is provided.
  • FileSystem (⭐37k) - FileSystem middleware for Fiber, special thanks and credits to Alireza Salary.
  • Limiter (⭐37k) - Rate-limiting middleware. Use to limit repeated requests to public APIs and/or endpoints such as password reset.
  • Logger (⭐37k) - HTTP request/response logger.
  • Monitor (⭐37k) - Monitor middleware that reports server metrics, inspired by express-status-monitor.
  • Pprof (⭐37k) - Pprof middleware that serves via its HTTP server runtime profiling data in the format expected by the pprof visualization tool.
  • Proxy (⭐37k) - Allows you to proxy requests to a multiple servers.
  • Recover (⭐37k) - Recover middleware recovers from panics anywhere in the stack chain and handles the control to the centralized ErrorHandler.
  • RequestID (⭐37k) - Adds a requestid to every request.
  • Session (⭐37k) - Session middleware. NOTE: This middleware uses our Storage package.
  • Skip (⭐37k) - Skip middleware that skips a wrapped handler is a predicate is true.
  • Timeout (⭐37k) - Adds a max time for a request and forwards to ErrorHandler if it is exceeded.
