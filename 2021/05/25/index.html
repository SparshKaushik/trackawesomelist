6. Awesome Ddd

Sample Projects / JVM languages

  • Eclipse CargoTracker - This project demonstrates how you can develop applications with the Jakarta EE platform using widely adopted architectural best practices like Domain-Driven Design (DDD). The code (⭐364) is intended to mirror a non-trivial application that developers in the real work would work on. It attempts to demonstrate first-hand how you can use Jakarta EE to effectively meet practical enterprise concerns such as productivity, agility, testability, flexibility, maintainability, scalability and security. The project is directly based on the well known original Java DDD sample application (⭐5.2k) developed by DDD pioneer Eric Evans' company Domain Language and the Swedish software consulting company Citerus.
