Jun 13 - Jun 19, 2022

Firmware

  • Sming Framework (⭐1.5k) - Superb C/C++ IoT Framework with support for ESP8266 and ESP32.

Tools

  • NodeMCU Flasher (⭐2.1k) - The official flashing tool for the NodeMCU OS.

Projects / Smart Home and IoT

  • ESPEasy (⭐3.4k) - Easily turn ESP modules into multifunction sensor devices for home automation systems.
  • DoorsignEPD (⭐100) - A smart doorsign with an E-Paper display using the ESP32.
  • HomePoint (⭐648) - Control MQTT/HomeKit smart home devices from an ESP32-powered screen.
  • openHASP - Control your home automation devices from a customizable touchscreen UI connected via MQTT.

Projects / LoRa

  • ESP32-Paxcounter (⭐1.9k) - Wifi & Bluetooth driven, LoRaWAN enabled, battery powered mini Paxcounter built on cheap ESP32 LoRa IoT boards.
  • Disaster Radio - A disaster-resilient communications network powered by the sun.

Projects / Music and Audio

  • Alles (⭐280) - A many speaker distributed music synthesizer using UDP multicast over WiFi, modeled after the alles machine/AMY.
  • ESP32-Radio (⭐1k) - Internet radio based on ESP32, VS1053 and a TFT screen.
  • ESPuino (⭐323) - RFID-controlled music player powered by ESP32.
  • Knobby (⭐193) - A handheld Spotify remote that encourages you to explore unfamiliar music.
  • PedalinoMini (⭐569) - A wireless MIDI pedal controller for guitarists, built with the ESP32.
  • Squeezelite-esp32 (⭐1.6k) - Streaming audio receiver with multi-room sync, AirPlay, Bluetooth, hardware buttons, display and more.
  • ThingPulse esp8266-spotify-remote (⭐271) - Control your Spotify player from a ESP8266 with color touch display.

Projects / Smartwatches

  • mutantW_V1 - An ESP32 based open source smartwatch with 1.7 inch display, WiFi, Bluetooth, NeoPixel and vibration.
  • StickWatch (⭐90) - A smartwatch module based on the M5Stick, using the ESP32.
  • Watchy - An open source e-paper watch with lots of options for customization.

Projects / Others

  • E-TKT (⭐430) - An ESP32 powered DIY label maker that mixes both old fashioned and contemporary technology.
  • FreeTouchDeck (⭐665) - Open source touch macropad and stream control deck with built-in web configurator.
  • SmartSpin2k (⭐227) - Transform your spin bike into a smart trainer with automatic resistance knob control in fitness apps like Zwift.
  • WirelessPrinting (⭐365) - Print wirelessly from Cura, PrusaSlicer or Slic3r to your 3D printer connected to an ESP module.
  • WLED - Control many types of RGB(W) LED strips with an ESP8266 or ESP32 over WiFi.

Libraries / Others

  • LVGL - An open-source graphics library providing everything you need to create embedded GUIs with easy-to-use graphical elements, beautiful visual effects and low memory footprint.
  • ESP32-audioI2S (⭐1.4k) - Plays mp3, m4a and wav files from SD card or stream via I2S interface.
  • HomeSpan (⭐1.9k) - A robust and extremely easy-to-use Arduino library for creating your own ESP32-based HomeKit devices.
  • LedFx (⭐1.6k) - A library for using audio input to create realtime light shows. LedFx can control multiple devices and works great with cheap ESP8266 nodes.

  • mJS (⭐2k) - A lightweight and restricted JS engine that is used by MongooseOS, compatible on the 32 and 8266.
  • ESPUI (⭐1k) - A simply library for making interactive web interfaces for both ESP's.
  • ESP32 ePaper (⭐232) - A full-featured library for using ePaper modules with the ESP32.
