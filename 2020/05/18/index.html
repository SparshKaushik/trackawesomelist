3. Awesome Transit

GTFS Libraries / C

  • CGTFS (⭐10) - C library for reading static GTFS feeds. Supports reading unpacked feeds into application memory or into SQLite databases.
  • RRRR Rapid Real-time Routing (⭐169) - RRRR (usually pronounced R4) is a C-language implementation of the RAPTOR public transit routing algorithm.

GTFS Libraries / Python

  • ESRI public-transit-tools (⭐183) - Tools for working with public transit data in ArcGIS (license for ArcGIS required).
  • gtfsdb (⭐171) - Python library for converting GTFS files into a relational database.
  • gtfsman (⭐22) - Repository-like tool in Python to manage and update a huge number of GTFS feeds.
  • gtfspy (⭐163) - Public transport network analysis and travel time computations using Python3. Compatible with Postgres/PostGIS, Oracle, MySQL, and SQLite. Used by gtfspy-webviz (⭐58).
  • Make GTFS (⭐12) - A Python library to make GTFS feeds from basic route information.
  • Mapzen GTFS (⭐30) - A Python GTFS library that supports reading individual GTFS tables, or constructing a graph to represent each agency in a feed.
  • multigtfs (⭐53) - A Django application to import and export GTFS.

GTFS Libraries / C#

  • GTFS Feed Parser (⭐71) - .Net/Mono implementation of a GTFS parser.

GTFS Libraries / Go

  • Go GTFS Parser (⭐44) - A GTFS parsing library for Go.

GTFS Libraries / PostgreSQL

  • gtfs-schema (⭐37) - PostgreSQL schema for GTFS feeds.

GTFS Libraries / Ruby

  • GTFS-viz (⭐90) - Ruby script that converts a set of GTFS files into a SQLite database + GeoJSONs (needed by the Transit Map (⭐361) web application)
