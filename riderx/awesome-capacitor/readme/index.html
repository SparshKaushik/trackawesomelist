Official plugins

Official plugins for Capacitor (⭐601) and maintained by ionic are listed here.

  • Action Sheet (⭐601) - Provides access to native Action Sheets.
  • App (⭐601) - Handles high-level App state and events.
  • App Launcher (⭐601) - Allows to check if an app can be opened and open it.
  • Browser (⭐601) - Provides the ability to open an in-app browser and subscribe to browser events.
  • Camera (⭐601) - Provides the ability to take a photo with the camera or choose an existing one from the photo album.
  • Clipboard (⭐601) - Enables copy and pasting to/from the system clipboard.
  • Device (⭐601) - Exposes internal information about the device, such as the model and operating system version, along with user information such as unique IDs.
  • Dialog (⭐601) - Provides methods for triggering native dialog windows for alerts, confirmations, and input prompts.
  • File system (⭐601) - Provides a NodeJS-like API for working with files on the device.
  • Geolocation (⭐601) - Provides simple methods for getting and tracking the current position of the device using GPS, along with altitude, heading, and speed information if available.
  • Haptics (⭐601) - Provides physical feedback to the user through touch or vibration.
  • Keyboard (⭐601) - Provides keyboard display and visibility control, along with event tracking when the keyboard shows and hides.
  • Local Notifications (⭐601) - Provides a way to schedule device notifications locally (i.e. without a server sending push notifications).
  • Motion (⭐601) - Tracks accelerometer and device orientation (compass heading, etc.).
  • Network (⭐601) - Provides network and connectivity information.
  • Preferences (⭐601) - Provides a simple key/value persistent store for lightweight data, ex Storage.
  • Push Notifications (⭐601) - Provides access to native push notifications.
  • Screen Reader (⭐601) - Provides access to TalkBack/VoiceOver/etc. and provides simple text-to-speech capabilities for visual accessibility.
  • Share (⭐601) - Provides methods for sharing content in any sharing-enabled apps the user may have installed.
  • Splash Screen (⭐601) - Provides methods for showing or hiding a Splash image.
  • Status Bar (⭐601) - Provides methods for configuring the style of the Status Bar, along with showing or hiding it.
  • Text Zoom (⭐601) - Provides the ability to change Web View text size for visual accessibility.
  • Toast (⭐601) - Provides a native notification pop up for displaying important information to a user.
