6. Awesome Snmp

Libraries / Python

  • gufo_snmp (⭐17) - The accelerated Python SNMP client library supporting both async and synchronous mode. It consists of a clean Python API for high-efficient BER parser and socket IO, implemented in the Rust language with PyO3 wrapper. Seems to be a bit early in the project's lifecycle, but it is easy to use and extremely fast, especially when querying many devices.
  • net-snmp Python bindings - Net-SNMP 5.4 and later contains Python bindings in the "python" sub-directory, though, they are not built by default.
  • easysnmp (⭐223) - A fork of net-snmp Python bindings that attempts to bring a more Pythonic interface to the library.
  • puresnmp (⭐88) - Pure Python3 SNMPv2 library without any dependencies.
  • snimpy (⭐189) - Snimpy is a Python-based tool providing a simple interface to build SNMP query.
  • python-netsnmpagent (⭐69) - This package allows to write net-snmp subagents in Python.
  • hnmp (⭐49) - HNMP is a high-level Python library to ease the pain of retrieving and processing data from SNMP-capable devices such as network switches, routers, and printers.
  • aiosnmp (⭐52) - Python package aiosnmp is an asynchronous SNMP client for use with asyncio. Only SNMP v2c is supported.
  • robotframework-snmplibrary (⭐12) - SNMPLibrary is a Robot Framework test library for testing SNMP.
  • Scapy (⭐12k) - Packet manipulation program & library. Scapy has a module (⭐12k) to build/dissect SNMP packets.

The review focuses on how cognitive science can provide forward models of human decision-making and inverse models of how humans think about others’ decision-making.
