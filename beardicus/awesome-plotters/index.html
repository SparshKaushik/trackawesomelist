  • svgo (⭐22k) - Node.js-based tool for optimizing SVG files.
  • Polargraph Optimizer (⭐57) - Optimize drawing plan for a polargraph.
  • penkit-optimize (⭐151) - An SVG optimizer that uses a vehicle routing solver to minimize plot time.
  • svg-crowbar (⭐840) - Chrome-only bookmarklet for extracting SVG from an HTML document.
