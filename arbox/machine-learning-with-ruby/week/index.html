Machine Learning Libraries / Frameworks

  • machine_learning_workbench (‚≠ê20) - Growing machine learning framework written in pure Ruby, high performance computing using Numo, CUDA bindings through Cumo (‚≠ê96). Currently implementating neural networks, evolutionary strategies, vector quantization, and plenty of examples and utilities.
  • Deep NeuroEvolution (‚≠ê125) - Experimental setup based on the machine_learning_workbench (‚≠ê20) towards searching for deep neural networks (rather than training) using evolutionary algorithms. Applications to the OpenAI Gym (‚≠ê36k) using PyCall (‚≠ê1.1k).

Machine Learning Libraries / Neural networks

  • machine_learning_workbench (‚≠ê20) - Framework including pure-Ruby implementation of both feed-forward and recurrent neural networks (fully connected). Training available using neuroevolution (Natural Evolution Strategies algorithms).

Machine Learning Libraries / Evolutionary algorithms

  • machine_learning_workbench (‚≠ê20) - Framework including pure-Ruby implementations of Natural Evolution Strategy algorithms (black-box optimization), specifically Exponential NES (XNES), Separable NES (sNES), Block-Diagonal NES (BDNES) and more. Applications include neural network search/training (neuroevolution).
